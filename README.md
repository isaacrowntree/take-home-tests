# Tennis Scoring Test

*Testing*

`npm test`

*Running*

`npm run scorer`

*Installing*

First, install NVM: https://github.com/nvm-sh/nvm

Then:

```
git clone git@github.com:isaacrowntree/take-home-tests.git
cd take-home-tests

git checkout dius-tennis

nvm install 14 # Latest node

npm install # Get Jest deps
```
